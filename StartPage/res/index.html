A Remote Compile Environment

BVRDE (Bjarke Viksoe's Remote Development Environment ) is an integrated development environment that allows you to edit, compile, debug and run C and C++ projects on a remote UNIX or LINUX server.
Integrated with the GNU C compiler and GDB debugger, it relieves you from ever issuing a make or debug command again.

What BVRDE Does for You

  • Neat syntax highlighting editor
  • Frees up your time to focus on the creative and challenging programming
  • Easy compilation of projects - on all platforms
  • Well-known debugging command and inspector support
  • Allows you to manage several remote projects in one solution
  • Is built using the top-quality WTL technology
